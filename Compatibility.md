# Introduction #

This page lists ROMs that are officially compatible with the randomizer. In time, ROM Hacks with partial or full support will be added here.

# Language Codes #
  * EN = English
  * JP = Japanese
  * FR = French
  * DE = German
  * IT = Italian
  * ES = Spanish
  * KR = Korean
